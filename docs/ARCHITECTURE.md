# AetherOS Architecture

## Overview

AetherOS follows Hexagonal Architecture with Domain-Driven Design principles.

## Project Structure

aetheros/
├── .github/           # GitHub configuration
├── api/               # API contracts
├── cmd/               # Entry points
├── application/       # Application layer
├── core/              # Core Framework
├── domain/            # DDD Domain
├── internal/          # Infrastructure
├── pkg/               # Public packages
├── plugins/           # Plugins
├── sdk/               # SDK
├── ui/                # UI
├── deploy/            # Deployment
├── docs/              # Documentation
├── specs/             # Specifications
├── rfcs/              # RFCs
├── adr/               # ADRs
├── examples/          # Examples
├── scripts/           # Scripts
├── tests/             # Tests
└── tools/             # Tools
